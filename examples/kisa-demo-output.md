# KISA 데모 출력 예시

## 공고 핵심 요약

확인된 공고 원문을 기준으로 직무, 지원자격, 우대사항, 전형 절차, 자기소개서 문항을 정리합니다.

## NCS/KSA 역량 매핑

Knowledge:

- 정보보호 기본 개념
- 네트워크와 웹 보안 이해
- 공공기관 디지털 서비스 이해

Skill:

- 취약점 진단
- 로그와 보안 이벤트 분석
- 보고서 작성과 개선안 도출

Attitude:

- 공공성
- 정확성
- 책임감
- 협업 태도

## 기관 분석

ALIO 주요사업, 기관 홈페이지, 보도자료를 근거로 최근 사업 방향을 정리합니다.

## 개선 과제

국회 지적사항과 경영평가 자료를 근거로 개선 과제를 정리하되, 면접에서는 비판보다 기여 방향으로 표현합니다.

## 자기소개서 소재

웹 보안 프로젝트 경험은 직무역량 문항에 우선 배치합니다.
기술 블로그 경험은 꾸준한 학습 태도와 지식 공유 태도에 연결합니다.

## 예상 면접 질문

1. 한국인터넷진흥원의 최근 주요 사업 중 본인이 관심 있는 사업은 무엇인가요?
2. 공공기관 보안 직무에서 가장 중요한 태도는 무엇이라고 생각하나요?
3. 웹 보안 프로젝트에서 발견한 문제를 어떻게 개선했나요?
4. 기관의 개선 과제에 본인이 기여할 수 있는 부분은 무엇인가요?
5. 마지막으로 하고 싶은 말이 있나요?
